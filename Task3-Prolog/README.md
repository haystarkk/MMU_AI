# Task 3: Prolog - Family Tree Program

## Overview
This task implements a family tree using Prolog, demonstrating logical programming capabilities including relationships like grandparents, parents, children, grandchildren, cousins, uncles, and aunts.

## Prerequisites
- SWI-Prolog installed on your system

## Installation Guide

### Install SWI-Prolog by operating system:

#### **Windows:**
```bash
1. Download from: https://www.swi-prolog.org/download/stable
2. Run swipl-latest-x64.exe
3. Follow installation wizard
4. Add to PATH: C:\Program Files\swipl\bin\
