# Task 4: Search and Optimization (BFS & DFS)

## Overview
This task implements Breadth First Search (BFS) and Depth First Search (DFS) algorithms to find paths from initial nodes to goal states.

## Files Included
- `bfs_search.py` - BFS implementation with shortest path finding
- `dfs_search.py` - DFS implementation (recursive and iterative)
- `graph_search_demo.py` - Combined demo with comparison
