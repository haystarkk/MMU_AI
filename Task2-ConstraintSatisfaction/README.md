# Task 2: Constraint Satisfaction Programming

## Overview
This task implements constraint satisfaction problems (CSP) for map coloring using backtracking algorithms.

## Part A: Australia Map Coloring
Color the regions of Australia using 3 colors (Blue, Red, Green) so that no two adjacent regions share the same color.

### Regions included:
- Western Australia (WA)
- Northern Territory (NT)
- South Australia (SA)
- Queensland (QLD)
- New South Wales (NSW)
- Victoria (VIC)
- Tasmania (TAS)

## Part B: Nairobi Sub-counties Coloring
Color Nairobi's 17 administrative sub-counties using the minimum number of colors possible.

### Sub-counties included:
Westlands, Dagoretti North, Dagoretti South, Langata, Kibra, Roysambu, Kasarani, Ruaraka, Embakasi North, Embakasi South, Embakasi Central, Embakasi East, Embakasi West, Makadara, Kamukunji, Starehe, Mathare
