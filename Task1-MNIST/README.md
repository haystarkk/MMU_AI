# Task 1: MNIST Dataset - Digit Classifier

## Overview
This program implements a Convolutional Neural Network (CNN) to recognize handwritten digits (0-9) from the MNIST dataset.

## Prerequisites
- Python 3.8 or higher
- pip package manager

## Installation Instructions

### Step 1: Install Python Dependencies

```bash
# Navigate to Task1-MNIST folder
cd Task1-MNIST

# Install required packages
pip install -r requirements.txt

# Or install individually
pip install tensorflow numpy matplotlib scikit-learn keras seaborn
